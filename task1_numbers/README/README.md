## 1. Опис алгоритму

1.  **Початок.**
2.  **Підключення бібліотек:** Підключити необхідні бібліотеки для роботи з консоллю (`iostream`) та файлами (`fstream`).
3.  **Ініціалізація змінних:**
    * Створити змінну `long long dob = 1;` для зберігання добутку. (Тип `long long` обрано, щоб уникнути переповнення, оскільки добуток може стати дуже великим).
    * Створити змінну `int number;` для тимчасового зберігання числа, зчитаного з файлу.
4.  **Відкриття файлу:**
    * Створити об'єкт вхідного файлового потоку `ifstream inputFile`.
    * Спробувати відкрити файл з назвою **`number.txt`** для читання.
5.  **Перевірка відкриття файлу:**
    * Перевірити, чи файл успішно відкрився (наприклад, `if (!inputFile)`).
    * **Якщо ні (помилка):** Вивести повідомлення про помилку (наприклад, "Не вдалося відкрити файл") і завершити програму з кодом помилки (`return 1`).
    * **Якщо так (успіх):** Продовжити виконання.
6.  **Цикл читання даних:**
    * Запустити цикл `while`, який триватиме, доки вдається зчитувати числа з файлу (`inputFile >> number`).
    * **Всередині циклу (для кожного числа):**
        * Перевірити умову: чи зчитане число (`number`) ділиться на 3 без залишку (`if (number % 3 == 0)`).
        * **Якщо так (кратне 3):** Оновити добуток, помноживши поточне значення `dob` на це число (`dob = dob * number;`).
        * **Якщо ні:** Нічого не робити, перейти до наступної ітерації циклу.
7.  **Кінець циклу.**
8.  **Закриття файлу:** Закрити файловий потік (`inputFile.close()`).
9.  **Аналіз результату та виведення:**
    * Перевірити, чи змінилося значення `dob`.
    * **Якщо `dob` залишився `1`:** Це означає, що у файлі не було знайдено жодного числа, кратного 3 (або файл був порожній). Вивести відповідне повідомлення (наприклад, "У файлі не було чисел, кратних 3").
    * **Якщо `dob` не дорівнює `1`:** Вивести на екран повідомлення та значення добутку (наприклад, "Добуток чисел, кратних 3, у файлі: [значення `dob`]").
10. **Кінець.**
