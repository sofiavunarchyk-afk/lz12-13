## 1. Опис алгоритму

1.  **Початок.**
2.  **Підключення бібліотек:** Підключити необхідні бібліотеки для роботи з консоллю (`iostream`), файлами (`fstream`), рядками (`string`), сортуванням (`algorithm`), математикою (`cmath`) та, за потреби, для налаштування консолі (`windows.h`).
3.  **Оголошення структури:**
    * Створити структуру `Student`, яка містить поля: `string PIB` (ПІБ), `string group` (номер групи) та `double avgMark` (середній бал).
4.  **Початок головної функції `main()`:**
5.  **Налаштування локалі:** (Опціонально) Налаштувати консоль для коректного відображення української мови (`SetConsoleCP(1251)`).
6.  **Ініціалізація масиву:**
    * Оголосити константу `N` (наприклад, `N = 3`).
    * Створити масив структур: `Student students[N]`.
7.  **Цикл введення даних (з консолі):**
    * Запустити цикл `for` від `i = 0` до `N - 1`.
    * **Всередині циклу:**
        * Вивести запит на введення даних ("Студент `i+1`:", "ПІБ:", "Група:", "Середній бал:").
        * Зчитати дані, введені користувачем, у відповідні поля `students[i].PIB` (використовуючи `getline` для ПІБ та групи), `students[i].group` та `students[i].avgMark` (використовуючи `cin`).
8.  **Сортування даних:**
    * Викликати функцію `std::sort` для масиву `students`.
    * Передати у функцію сортування лямбда-вираз, який порівнює двох студентів (`a` і `b`) за полем `avgMark` за спаданням (`return a.avgMark > b.avgMark;`).
9.  **Відкриття файлу для запису (Модифікація):**
    * Створити об'єкт вихідного файлового потоку `ofstream fout`.
    * Спробувати відкрити файл з назвою **`results.txt`** для запису (`fout.open("results.txt")`).
10. **Перевірка відкриття файлу:**
    * Перевірити, чи файл успішно відкрився (`if (!fout)`).
    * **Якщо ні (помилка):** Вивести повідомлення про помилку на консоль (`cout`) і завершити програму (`return 1`).
    * **Якщо так (успіх):** Продовжити.
11. **Виведення та запис заголовку таблиці:**
    * Вивести заголовок таблиці ("П.І.П.", "Група", "Середній бал") на консоль (`cout`).
    * **(Модифікація)** Записати *той самий* заголовок таблиці у файл `results.txt` (`fout`).
12. **Цикл виведення та запису даних:**
    * Ініціалізувати логічну змінну `bool found = false;`.
    * Запустити цикл `for` від `i = 0` до `N - 1`.
    * **Всередині циклу:**
        * Перевірити умову: `if (students[i].avgMark > 4.0)`.
        * **Якщо так:**
            * Встановити `found = true;`.
            * Відформатувати (вирівняти, задати ширину) та вивести дані `students[i]` (ПІБ, група, бал) на консоль (`cout`).
            * **(Модифікація)** Відформатувати та записати *ті самі* дані `students[i]` у файл `results.txt` (`fout`).
13. **Виведення та запис підсумку:**
    * Вивести нижню межу таблиці на консоль (`cout`).
    * **(Модифікація)** Записати нижню межу таблиці у файл (`fout`).
    * Перевірити `if (!found)` (якщо жодного студента не було знайдено).
    * **Якщо так:** Вивести повідомлення "Немає студентів..." на консоль (`cout`) та **(Модифікація)** записати *те саме* повідомлення у файл (`fout`).
14. **Закриття файлу:**
    * Закрити файловий потік (`fout.close()`).
    * (Опціонально) Вивести на консоль повідомлення про успішний запис у файл.
15. **Кінець** (`return 0`).
